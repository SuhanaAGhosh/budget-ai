# budget-ai

An AI-powered personal budgeting assistant.

## What it does
- Takes a CSV of transactions (bank/UPI statement)
- Auto-categorizes each transaction using an LLM
- Shows a dashboard of spending trends
- Lets you ask questions in plain English: "how much did I spend on food in June?"

## Stack
-Python Oriented
- Python + FastAPI (backend)
- SQLite (database)
- Claude API (LLM)
- Streamlit (dashboard)
